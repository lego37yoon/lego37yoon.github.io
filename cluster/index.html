본 HTML 웹 문서는 테스트용이며, 웹 자체는 Chrome에서 실행될 것을 상정하고 만들었습니다. 하지만 저는 불여우 파죠.

그러므로 우리 모두 불여우를 써요!

당신이 불여우를 써야하는 이유!

  • Firefox Quantum으로 속도가 이전에 비해 훨씬 빨라졌습니다!
  • Firefox Sync로 iOS, Android, Windows, macOS는 물론이고 Linux 및 UNIX-liked환경에서 이용해 보세요
  • PC환경 뿐만 아니라 안드로이드 환경에서도 동일한 설정으로 부가기능을 사용할 수 있습니다.
  • Firefox는 Chrome에 비해 더 자유로운 환경을 담보합니다: 다양한 부가기능, 개조에 가까운 꾸미기, 그리고 PRISM 프로젝트에서 자유로운 웹 브라우징.
  • 웹 화면을 자유롭게 갈무리 할 수 있고 업로드도 가능한 Firefox Screenshots를 제공합니다. 별도로 알캡쳐나 오픈캡쳐, 픽픽같은 도구를 설치할 필요가 없죠.
  • 개발 및 번역, 도움에 직접 참여할 수 있습니다. 모질라 한국 커뮤니티에 가입하지 않아도 MDN이나 Mozilla Support에서 번역가로 활동할 수 있습니다.
