오늘의 수업: 도장찍기

오늘은 도장 찍기를 해봅니다.

  1. head 안 style 태그에 다음과 같이 지정합니다.
#canvas { width:100%; height:1000px; }
  1. 스크립트에 다음과 같이 작성합니다. clientX/Y는 문서 기준, offsetX/Y는 오브젝트 기준 위치, 그리고 screenX/Y는 모니터 기준 위치입니다.
function stamp() { //도장을 찍는 코드 //자바스크립트로 태그 동적 생성 var img = document.createElement("img"); //<img src="pika.jpg">를 자바스크립트로! img.src = "./pika.jpg"; img.style.position="absolute"; img.style.left= event.clientX; img.style.top= event.clientY; var canvas = document.getElementById("canvas"); canvas.appendChild(img); //붙여라! img.style.width="50px"; }
  1. 도장을 찍을 영역을 설정합니다.
<div id="canvas" onMouseDown="stamp()"></div>
