자바스크립트가 좋은 점!

  • 고급 언어이다!

    • 자바스크립트는 고급 언어이기 때문에 사용자가 하지 않아도 알아서 처리해주는 부분이 많다.

  • 사용하기 쉽다.

    • 어렵지 않은 문법으로 초심자도 쓰기 쉽다.

  • 구동하기도 쉽다

    • 컴파일 없이 웹 브라우저에 던져주면 쓸 수 있다.

    변수에 대해 알아보자!

    변수에는 여러가지를 저장할 수 있는데, 이 변수는 디스크가 아닌 RAM에 저장된다.

    자바스크립트에서의 변수

    변수 선언하려면

    var x;

    변수 값을 정해보자(예시: 1을 값으로 정해보자.

    var x= 1;

    변수 값을 변경한다면?(1로 처음 정하고 다시 2로 값 변경)

    var x= 1;
    x = 2;

    예시

    어떻게 했냐면...

    첫번째: x의 내용 출력

                    var x = 1;
                alert(x);

    두번째: 숫자끼리 더해서 출력

                    var x = 2;
                alert(x+1);

    세번째: 문자 출력

                    var x = 1;
                alert("사람들이 많네요.");

    네번째: 문자 +1

                    var x = "1";
                alert(x+1);

    다섯 번째: 숫자 계산

    연산자: * 곱하기, + 더하기, / 나누기, % 나눈 나머지, - 빼기

    여섯 번째: 한 번 선언한 var를 두 번 선언하지 않는다.

    일곱 번째: 플플과 마마

                    var x = 1;
                alert(x++)
  • ++이 앞에 있으면 더하고 저장한 후 출력, ++이 뒤에 있으면 출력하고 더함.(변수에) -- 또한 같은 방식으로 움직인다!

    • 여덞 번째: 대입연산자

    +=, -=, *=, %=, /=

                    var x = 1;
                alert(x += 1); // 1만큼 더함

    아홉 번째: 논리 연산자

    ==, >=, <=, !=(같지 않니?), <, > 

                    var x = 1;
                var y = 3;
                alert(x != y);

    이 경우 true라는 결과 값이 나타난다.

    만약 변수를 문자로 설정할 경우 ==과 !=만 가능하다. 둘 중 하나만 설정하면 아예 작동하지 않는다.













    그림을 움직이려면?

    <button onclick="move()">피카츄, 너로 정했다!</button> <br> <img id="pikachu" src="pika.jpg" style="position:absolute; left:100px;"> <script> function move(){ document.getElementById("pikachu").style.left = "200px"; } </script>

    그림을 버튼 누를 때마다 움직이게 하고 싶다면

    전역변수를 이용한다. function 앞에서 변수를 선언하자.

    <button onclick="move()">피카츄, 너로 정했다!</button> <br> <img id="pikachu" src="pika.jpg" style="position:absolute; left:100px;"> <script> var x = 100; function move(){ x+=100; document.getElementById("pikachu").style.left = x+"px"; } </script>

    하려면...

    <input id="input" placeholder="원하는 숫자를 입력해 주세요!"></input> <button onclick="check()">1인지 물어보기!</button> <script> function check(){ var z = document.getElementById("input").value; if(z==1){ alert("1이다!"); } else { alert("1이 아니잖아!"); } } </script>












    하려면...

    <button onclick="move2()">피카츄, 너로 정했다!</button> <br> <img id="pikachu2" src="pika.jpg" style="position:absolute; left:100px;"> <br><br><br><br><br><br><br><br><br><br><br> <script> var x =100; var y = 0; function move2(){ if(x<=400 && y==0){ x += 100; document.getElementById("pikachu2").style.left = x+"px"; } else if(x==100&&y==1){ y=0; x += 100; document.getElementById("pikachu2").style.left = x+"px"; } else if(x==500){ y =1; x -= 100; document.getElementById("pikachu2").style.left = x+"px"; } else { x -= 100; document.getElementById("pikachu2").style.left = x+"px"; } } </script>
